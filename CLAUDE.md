# Jackbox Minigame

## Code Style

See [STYLE_GUIDE.md](./STYLE_GUIDE.md) for coding conventions.

The short version:
- Arrow functions only. No `function` keyword.
- No semicolons. No else. No braces for one-liners.
- Early returns. Let TypeScript infer.
- Fail fast. No fallbacks. Let it crash.
- Code as haiku.
